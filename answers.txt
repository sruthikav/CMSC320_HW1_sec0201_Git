Q1: Why might we consider using Git?

Answer: Git is mostly used for version control. It makes it easy to collaborate,
keep tracking of what changes are made and when, as well as merging various
versions of code and projects.

Q2: What field could data science be applied to? Give an Example.

Answer: Data science can be applied to any field. One example of an application of
data science is in the medical field. In the medical field, it can be used to 
analyze large formats of patient data. This analysis can lead to predicting risk
for diseases, various symptoms, and help doctors make informed decisions.

Q3: What is the difference between Merging and Rebasing?

Answer: The difference between merging and rebasing is that merging combines
two branches and creates a new commit with the new version. Alternatively,
rebasing moves commits from one branch to a different branch. This is much
cleaner.
